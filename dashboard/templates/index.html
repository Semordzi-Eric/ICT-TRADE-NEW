How It Works
🔎 Detection

FVG + Order Blocks + Liquidity Sweeps + BOS + CHoCH on M15 bars — all 16 markets simultaneously via threads.

🧠 ML + Intuition

LightGBM + XGBoost + LSTM ensemble. When ML probability < 0.65, Intuition Mode scores 9 confluence factors and can still fire a trade.

🛡️ Risk Management

Adaptive sizing based on rolling win rate. News blackout 30min around high-impact events. Correlation filter prevents double exposure.

Quick Start Guide

1. Open MetaTrader 5 on your desktop and log in to your account

2. Click Connect above — the bot attaches to the running terminal

3. Go to Data → Select all symbols → Download (3yr recommended)

4. Go to Train → Train All Markets → wait for models to be saved

5. Go to Benchmark → Review leaderboard → confirm models are performing

6. Go to Live Trading → Select symbols → Enable Intuition Mode → Start

⚠️ Always test on a demo account first. Never risk money you cannot afford to lose.
